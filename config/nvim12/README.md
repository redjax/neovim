# Neovim 0.12.0 Configuration

I am porting my [main Neovim configuration](../nvim) to [Neovim 0.12.0](https://neovim.io/doc/user/news-0.12/). The biggest change is removing the Lazy package manager in favor of the new built-in [`vim.pack`](https://neovim.io/doc/user/pack/).

See the [Plugin docs](../../docs/PLUGINS.md) for documentation on converting plugin specs from Lazy to `vim.pack`.

Progress on the configuration port is tracked in [issue #444](https://github.com/redjax/neovim/issues/444).
